A responsive PC Build and Customization Website developed using HTML, CSS, Bootstrap, JavaScript, Spring Boot, and Oracle Database. Users can customize PC components, calculate total price, and place orders online.

# PC Factory - Custom PC Builder Website

## 📌 Project Overview

PC Factory is a responsive web application that allows users to build and customize their own PC by selecting various hardware components. The system automatically calculates the total price based on selected components and enables users to place orders through an integrated backend system.

## 🚀 Features

- Custom PC Configuration
- Component Selection
- Dynamic Price Calculation
- Responsive User Interface
- Order Placement System
- Backend API Integration
- Oracle Database Connectivity

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Hibernate

### Database
- Oracle SQL Developer


## ⚙️ Functional Modules

### Home Page
Displays featured products, offers, and company information.

### PC Builder
Allows users to select:
- Processor
- Motherboard
- RAM
- Graphics Card
- Storage
- Power Supply
- Cabinet

### Price Calculator
Automatically updates total price based on selected components.

### Order Management
Stores customer orders in Oracle Database through Spring Boot REST APIs.

## 🗄️ Database

The project uses Oracle Database to store:

- Customer Details
- Selected Components
- Order Information
- Total Price
